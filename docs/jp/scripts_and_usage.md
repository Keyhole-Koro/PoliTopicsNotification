# スクリプトと使用法 (PoliTopicsNotification)
[English Version](../../docs/scripts_and_usage.md)

このドキュメントでは、Notification パッケージの実行可能なスクリプトと一般的なワークフローをリストします。
パスは `PoliTopicsNotification` からの相対パスです。

## NPM スクリプト
- `npm run build`: TypeScript を `dist/` にコンパイルします。
- `npm run test`: Jest テストを実行します。
- `npm run prepare`: 公開/インストールの前にビルドします。

## ユースケース

### パッケージをビルドする
```bash
npm run build
```

### テストを実行する
```bash
npm run test
```
